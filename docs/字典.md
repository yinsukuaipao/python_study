# 字典

字典是一系列键-值对

## 创建字典

```python
# d = {'k1':1,'k2':2}
d = {
    'k1':1,
    'k2':2
    }
print(d) # {'k2': 2, 'k1': 1}
```

## 访问字典中的值

```python
d = {'k1':1,'k2':2}
print(d['k1']) # 1
print(d['k2']) # 2
```

## 添加键-值对

```python
d = {}
d['k1'] = 1
d['k2'] = 2
print(d) # {'k2': 2, 'k1': 1}
```

## 修改字典中的值

```python
d = {'k1':1,'k2':2}
d['k1'] = 3
print(d) # {'k2': 2, 'k1': 3}
```

## 删除键-值对

```python
d = {'k1':1,'k2':2}
del d['k1']
print(d) # {'k2': 2}
```

## 遍历字典

### 遍历所有键-值对

```python
d = {'k1':1,'k2':2}
for k, v in d.items():  # 不加items()也可以
    print("k:" + k + " v:" + str(v))

# k:k2 v:2
# k:k1 v:1

print(d.items()) # [('k2', 2), ('k1', 1)]
```

### 遍历key值

```python
d = {'k1':1,'k2':2}
for k in d.keys(): # 不加keys()也可以
    print("k:" + k)

# k:k2
# k:k1
```

### 顺序遍历key值

```python
d = {'k1':1,'k2':2}
for k in sorted(d.keys()):
    print("k:" + k)

# k:k1
# k:k2
```
