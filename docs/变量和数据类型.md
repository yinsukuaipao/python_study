### 字符串方法

#### 改变每个单词的首字母为大写

```python
str = "aa bb"
print(str.title()) #Aa Bb
```

#### 全部为大写

```python
str = "aa bb"
print(str.upper()) #AA BB
```

#### 全部为小写

```python
str = "AA BB"
print(str.upper()) #aa bb
```

#### 合并字符串

```python
str1 = "aa"
str2 = "bb"
str = str1 + " " + str2
print(str) #aa bb
```

#### 删除空白（空格 制表符 换行等）

```python
str = " python "
#删除末尾空白
str.rstrip() #" python"

#删除开头空白
str.lstrip() #"python "

#删除字符串两端空白
str.strip() #"python"
```

## 列表（数组）

### 声明定义

```python
arr = ['a','b','c']
print(arr) #['a', 'b', 'c']
print(arr[0]) #a
```

### 添加元素

#### 添加到末尾（append）

```python
arr = []
arr.append('d')
print(arr) #['d']
```

#### 添加到任意位置（insert）

```python
str = ['a', 'b', 'c']
str.insert(0, 'd')
print(str) #['d', 'a', 'b', 'c']
```

### 删除元素

#### 按索引删除元素（del）

```python
str = ['a','b','c']
del str[0]
print(str) #['b', 'c']
```

#### 删除列表元素并返回（pop）

```python
str = ['a','b','c']
v = str.pop()
print(v) #c
print(str) #['a', 'b']
```

#### 删除列表任意元素并返回（pop）

```python
str = ['a','b','c']
v = str.pop(1)
print(v) #b
print(str) #['a', 'c']
```

#### 根据值删除元素（remove）

```python
str = ['a','b','c']
str.remove('a')
print(str) #['b', 'c']
```

### 列表长度（len）

```python
str = ['a','b','c']
print(len(str)) # 3
```

### 排序

#### 永久排序（sort）

```python
str = ['a','c','b']
str.sort()
print(str) #['a', 'b', 'c']
str.sort(reverse=True)
print(str) #['c', 'b', 'a']
```

#### 临时排序（sorted）

```python
str = ['c','a','b']
print(sorted(str)) #['a', 'b', 'c']
print(str) #['c', 'a', 'b']
# 同样支持reserse=True
```

#### 反序（reverse）

```python
str = ['c','a','b']
str.reverse()
print(str) #[['b', 'a', 'c']
```

### 创建数字列表

#### range

```python
print(range(1,5)) #[1, 2, 3, 4]
for v in range(1,5):
    print(v)

#1
#2
#3
#4

# 用list()转化为数组，可能类似于类型转换，但是range返回的用不用list，print和for的结果是一样的
print(list(range(1,5))) # [1, 2, 3, 4]

print range(3) # [0, 1, 2]

# 设置步长
num = range(2, 11, 2)
print(num) #[2, 4, 6, 8, 10]
```

### 求数字列表中最大值（max）

### 求数字列表中最小值（min）

### 列表解析

```python

```

### 切片

```python
num = [1,2,3]
print(num[1:2]) # [2]
print(num[1:])  # [2, 3]
print(num[:2])  # [1, 2]
print(num[:])   # [1, 2, 3]
print(num[-2:]) # [2, 3]
```

### 复制列表（拷贝）

```python
l1 = [1,2,3]
l2 = l1[:]   # 值拷贝
l3 = l1      # 引用拷贝
```

## 元组

元组看起来像列表，使用圆括号。定义元组后，可以使用索引来访问元素，但是不能修改。

```python
t = (1,2)
print(t[0]) # 1
for v in t:
    print(v)

# 1
# 2

t2 = (2,3)
print(t2) # (2, 3)
```
