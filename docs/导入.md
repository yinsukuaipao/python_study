## 导入类

### 导入单个类

```python
from modulename import classname
```

### 同一个模块多个类

```python
class Car():

class Battery():

```

### 从一个模块导入多个类

```python
from modulename import classname1, classname2
```

### 导入整个模块

```python
import modulename
```

### 导入模块所有类

```python
from modulename import *
```
